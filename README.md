# deno-kv-sparql

Execute SPARQL queries on Deno Kv storage.

## Develop

Run unit tests.

```sh
deno task test
```

Format the code.

```sh
deno fmt
```

Update outdated dependencies to their latest versions.

```sh
deno task outdated
```
